Metodologia

Magic Formula — Joel Greenblatt ("The Little Book That Beats the Market", 2005). Objetivo: encontrar boas empresas a preços baratos de forma sistemática.

NEUTRO: CAGR Lucros negativo · Dív/EBIT > 3x · margem < 7%.
